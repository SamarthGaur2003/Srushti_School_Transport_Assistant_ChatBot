# Srushti School Transport Assitant ChatBot
Project Report File of created Chatbot through DialogFlow which answers some common FAQs about school bus transport services.

This project is an AI-based chatbot developed using Google Dialogflow ES for Srushti School Transport Services.

## Features
- School Information
- Pickup Timings
- Drop Timings
- School Timings
- Bus Fee Information
- Contact Information
- Fallback Response Handling

## Technology Used
- Google Dialogflow ES
- NLP (Natural Language Processing)
- GitHub
